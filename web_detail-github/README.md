# LINSY 商城 WEB 端交易链路 Demo

## 打开方式（重要）

### 方式 A：单文件（推荐，不会空白）
双击 **`standalone.html`** — 所有代码已内嵌，无需其他文件。

### 方式 B：多文件
1. **先解压 zip**，不要从压缩包内直接打开
2. 双击 `index.html`
3. 确保以下文件在同一目录：
   - index.html
   - mall-trade-demo.js
   - mall-trade-demo.css
   - mall-i18n.js

## GitHub Pages

上传 **`web_detail-github` 文件夹内全部文件** 到仓库根目录。

访问：`https://<用户名>.github.io/<仓库名>/`

或上传 `standalone.html` 并访问：`https://<用户名>.github.io/<仓库名>/standalone.html`

## 功能

- 商品详情 / List Mode 右侧抽屉 / 意向清单 / PO / 发运 / 付款
