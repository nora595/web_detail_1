【重要】上传到 GitHub 仓库根目录，不要放进子文件夹！

正确结构（6 个文件与 README 同级）：
  web_detail/
    index.html          ← 必须是这个，不是 README
    mall-trade-demo.js
    mall-trade-demo.css
    mall-i18n.js
    .nojekyll
    README.md

错误做法（会导致首页空白）：
  web_detail/
    web_detail-github/   ← 不要这样
      index.html
      ...

上传步骤：
1. 解压本 zip
2. 选中解压出来的 6 个文件（不是文件夹）
3. GitHub 仓库 → Add file → Upload files → 拖到根目录
4. 若已有 web_detail-github 子文件夹，可删除后重新上传

访问地址：https://你的用户名.github.io/web_detail/
